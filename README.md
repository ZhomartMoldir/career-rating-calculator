# career-rating-calculator
Career guidance rating calculator for students


A Windows Forms application (C#) that helps students evaluate their university admission profile.

## How it works
- Select your achievements across 8 categories: olympiads, internships, hackathons, debates, sports, publications, leadership, volunteering
- Each category is weighted by level (school → international) and result (participant → winner)
- Enter your UNT, SAT, IELTS, and GPA scores
- The calculator outputs a total score and places you in one of four tiers: Basic, Average, High, or Top

## Tech
- C# / Windows Forms / Visual Studio
